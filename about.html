About Code.iBlogger

Code.iBlogger is such a platform where a student can find and learn programming concept and other subjects of Computer Science. In other way we can say Code.iBlogger is a note sharing platform where a student can learn programming and other computer subjects.

Aim

We prioritized user experience and enhanced design of this website such as navigating pages and subjects, colored text inside code box. Code.iBlogger attemps to solve the problem of students who don't know how to code a program or understand mathematical and logical problems.

Facility

Code.iBlogger provides simplest concept of programming language. Not only programming languages but also other subjects of computer science are available.

Subjects :- C | Digital Logic | C++ | Data Structures | Java | Discrete Math | Database Systems | Computer Networks | Operating Systems | Computer Graphics etc.

1. Easy-to-Understand Notes

Basically all student want to avail all notes corresponding to their course. So here in Code.iBlogger, notes are easily available in the theory section of a subject. Notes are written in simple and easy language.

2. Examples & Practices

Here, practice set and examples are given in each topic. After reading one can practice it. Also there are some quizzes to solve and improve programming skills.

3. Colorful Codes

Unlike handwritten notes, Code.iBlogger is designed in such a way that the programs look like vs-code type. A program consists of colorful token according its syntax.

4. Downloadable Practical Files

Anyone who needs files of source code can download easily. There are 2 type of files. One is normal notepad file. Another is respected to programming language.

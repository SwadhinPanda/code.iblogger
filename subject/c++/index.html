Object-Oriented Paradigm

Object-Oriented Programming treats data as important element in the program development. OOP consists of a number of entities called Objects. The data of an object can be accessed only by the methods associated with that object.

An object is considered to be partitioned area of computer that stores data and a set of operations that can access the data. Some features of Object-Oriented Paradigm are:

  1. Emphasis is on data rather than procedure
  2. Program divided into Objects
  3. Data Structures are designed such that they characterize the objects
  4. Methods that operate on the data of an object are tied together in the data structures
  5. Data is hidden and can't be accessed by external functions
  6. Objects may communicate with each other through methods
  7. New data and members can be easily added whenever necessary.
  8. Follows bottom-up approach

Run time polymerphism is the time period when a syntax correct program is actucally executed in the computer system.

Tokens in C

In C programming, a token is the smallest individual unit of a program. A 'C' program consists of various types of tokens, which can be classified into the following categories:

Keywords: Keywords are reserved words that have a specific meaning in the C language. Examples include if, else, for, while, return, and void.

Identifiers: Identifiers are names given to variables, functions, and other entities in a C program. An identifier can consist of letters, digits, and underscores. It must start with a letter or an underscore. Examples of identifiers include sum, i, j, and my_function.

Constants: Constants are fixed values that cannot be changed during program execution. C supports various types of constants such as integer constants, floating-point constants, character constants, and string constants. Examples include 42, 3.14, 'a', and "Hello World".

Operators: Operators are symbols that perform mathematical or logical operations on operands. C supports various types of operators such as arithmetic operators, relational operators, logical operators, and bitwise operators. Examples include + - * / < > && || &

Special symbols: These are used to indicate the beginning or end of a statement, block of code, or function. Examples include , ; { } ( )

It is important to note that whitespace (spaces, tabs, and line breaks) is not considered a token in C programming.

Datatypes in C

Datatypes specify the type of data that a variable can hold. C supports a variety of data types, including:

int: These data types are used to store integer values, which can be positive, negative, or zero. C supports four integer data types: char (1 byte), short (2 bytes), int (2 or 4 bytes), and long (4 or 8 bytes).

float: These data types are used to store real numbers with decimal places. C supports two floating-point data types: float (4 bytes) and double (8 bytes).

char: The char data type is used to store a single character value, such as a letter or digit. It takes up 1 byte of memory.

In addition to these basic data types, C also allows for the creation of user-defined data types, such as structures and enums.

Operators in C

In C programming, operators are symbols or keywords that perform various operations on data. Operators in C can be classified into the following categories:

Arithmetic operators: These operators perform mathematical operations such as addition, subtraction, multiplication, division, and modulus (remainder). Examples of arithmetic operators in C include +, -, *, /, and %.

Relational operators: These operators compare two values and return a Boolean value of true or false. Examples of relational operators in C include <, >, <=, >=, ==, and !=.

Logical operators: These perform logical operations on Boolean values and return a Boolean value of true or false. Examples of logical operators in C include && (logical AND), || (logical OR), and ! (logical NOT).

Bitwise operators: These operators perform operations on individual bits of binary numbers. Examples of bitwise operators in C include & (bitwise AND), | (bitwise OR), ^ (bitwise XOR), and ~ (bitwise NOT).

Assignment operators: These operators are used to assign values to variables. Examples of assignment operators in C include =, +=, -=, *=, /=, and %=.

Increment and decrement operators: These are used to increment or decrement the value of a variable. Examples of these operators in C include ++ (increment), and -- (decrement).

Conditional operator: The conditional operator is a shorthand way of writing an if-else statement. It takes three operands and returns one of two values based on a Boolean condition. The syntax for the conditional operator in C is condition ? value_if_true : value_if_false.

Structure of a C++ Program

The structure of a C++ program typically consists of the following components:

Preprocessor Directives: These are special instructions to the preprocessor to include header files that provide declarations of functions and macros. Preprocessor directives are placed at the beginning of the program and start with the "#" symbol.

Global Declarations: These are declarations of variables, functions, and objects that will be used throughout the program. Global declarations are placed after the namespace declarations.

Function Declarations: These are declarations of functions that will be defined later in the program. Function declarations are placed after the global declarations.

Function Definitions: These are the actual implementations of the functions declared earlier. Function definitions are placed after the function declarations.

Main Function: It is the entry point of the program, where the program execution begins. The main function is required in every C++ program and is defined as follows: 

int main() {
// program statements
return 0;
}

Statements: These are the actual instructions that perform the tasks of the program. Statements are placed inside the main function. Here's an example program that demonstrates the structure of a C++ program: 

// Preprocessor Directive
#include <iostream.h>

// Global Declaration
int x = 42;

// Function Declaration
int add(int a, int b);

// Function Definition
int add(int a, int b) {
  return a + b;
}

// Main Function
int main() {
  // Statements
  int y = 10;
  int z = add(x, y);
  cout << "The sum of " << x << " and " << y << " is " << z << endl;
  return 0;
}

In above program, the preprocessor directive "#include <iostream>" is used to include the standard input/output library, the global declaration "int x = 42;" declares a global variable, the function declaration "int add(int a, int b);" declares a function, the function definition "int add(int a, int b) { return a + b; }" defines the function, and the main function contains the statements that perform the task of the program.

Expression and Control Structure

An expression is a combination of literals, variables, operators, and function calls that evaluate to a value. For example, the following is an expression in C++: 

int x = 5;
int y = 10;
int z = x + y;

Control structures are used to control the flow of execution in a program. There are three types of control structures in C++: 

 int x = 10;
 if (x > 5) {
 cout << "x is greater than 5" << endl;
}
Loop

Loops allow you to execute a block of code repeatedly. The most common types of loops are the for loop, the while loop, and the do-while loop. Here's an example of a for loop: 

 for (int i = 0; i < 5; i++) {
   cout << i << endl;
 }

In above example, the loop will execute 5 times, with the variable i taking on the values 0, 1, 2, 3, and 4.

Jump

Jump statements allow you to transfer control to a different part of the program. The most common types of jump statements are break, continue, and return. Here's an example of a break statement inside a loop: 

for (int i = 0; i < 5; i++) {
  if (i == 3) {
    break;
  }
  cout < i < endl;
}

In above example, the loop will execute 3 times, with the variable i taking on the values 0, 1, and 2. When i is equal to 3, the break statement is executed, causing the loop to terminate early.

Function in C++

Functions are a fundamental part of programming in C++. A function is a block of code that performs a specific task, and can be called from other parts of the program as needed. In C++, functions are declared using a function prototype, and defined using a function definition.

Here's an example of a simple function that takes two arguments and returns their sum: 

int sum(int a, int b) {
  return a + b;
}

In above example, the function is named sum, and takes two arguments of type int. Inside the function, the arguments are added together, and the result is returned using the return keyword.

Functions can be called from other parts of the program using their name, and passing the necessary arguments. For example: 

int x = 5;
int y = 10;
int z = sum(x, y);

In above example, the sum function is called with the values of x and y, and the result is stored in the variable z.

Functions can also be declared with default arguments, which are used when the function is called without providing a value for that argument. For example: 

int multiply(int a, int b = 2) 
{ return a * b; }

In above example, the multiply function takes two arguments, with a default value of 2 for the second argument. If the function is called with only one argument, the second argument will default to 2.

C++ also supports function overloading, which allows you to define multiple functions with the same name, but different parameters. For example: 

 int sum(int a, int b) 
 { return a + b; }
 double sum(double a, double b)
 { return a + b; }

In above example, the sum function is defined twice, once for int arguments, and once for double arguments. When the function is called, the appropriate version is automatically selected based on the types of the arguments.

Functions are a powerful tool for organizing and structuring code in C++, and are used extensively in real-world applications.

Type of Functions

In C++, there are several types of functions based on their return type, arguments, and purpose. Here are the most common types of functions in C++:

Void Function: A void function is a function that does not return any value. It is commonly used when the primary purpose of the function is to perform some action or side-effect, such as printing output to the console or modifying global variables.

Functions with Return Type: A function with return type is a function that returns a value of a specified data type. These functions are used when the primary purpose of the function is to perform some calculation or operation and return a value to the calling code.

Functions with Arguments: A function with arguments is a function that takes one or more parameters or arguments. These parameters are inputs to the function and can be used inside the function to perform some computation. Arguments can be passed by value, by reference or by pointer.

Inline Functions: An inline function is a function that is expanded in-line at the point of use, rather than being called as a separate function. This can lead to faster execution times, since the overhead of function calls is eliminated. Inline functions are typically used for small, frequently called functions.

Overloaded Functions: An overloaded function is a function that has the same name as one or more other functions, but with different parameters. The correct function to call is determined based on the number and types of the arguments provided.

These are just a few of the common types of functions in C++. By using these different types of functions, you can create powerful and flexible programs that can perform a wide range of tasks

Defining Member Function

In C++, member functions are functions that are defined inside a class and can access the properties and methods of that class. Member functions can be defined in the class declaration itself or in a separate implementation file.

Here's an example of a class definition with a member function defined in the class declaration: 

class Rectangle {
  private:
    int width, height;
  public:
   void set_values(int w, int h)
    {
      width = w;
      height = h;
    }
    int area() {
      return width * height;
    }
};

In above example, the Rectangle class has two private properties (width and height) and two public member functions (set_values and area). The set_values function is used to set the values of the width and height properties, while the area function calculates and returns the area of the rectangle.

Member functions can also be defined outside of the class declaration, in a separate implementation file. Here's an example of a Rectangle class declaration in a header file (Rectangle.h) and its implementation in a source file (Rectangle.cpp):

Rectangle.h:

class Rectangle {
  private:
    int width, height;
  public:
    void set_values(int w, int h);
    int area();
};

Rectangle.cpp:

#include "Rectangle.h"

void Rectangle::set_values(int w, int h) 
{
  width = w;
  height = h;
}

int Rectangle::area() {
  return width * height;
}

In above example, the set_values and area member functions are defined in the Rectangle.cpp source file using the scope resolution operator ::. The Rectangle.h header file is used to declare the class and its member functions, which are then implemented in the source file.

Defining member functions in C++ allows you to encapsulate the behavior of an object within the object itself, making it easier to understand and maintain your code. It also allows you to create reusable code, since you can create multiple objects of the same class and use the same member functions to manipulate them.

Constructors

In C++, a constructor is a special member function of a class that is called automatically when an object of that class is created. The constructor is used to initialize the properties of the object and perform any other setup that is needed before the object can be used.

Here's an example of a class with a constructor:

 class Person {
   private:
     string name;
     int age;
   public:
     Person(string n, int a) 
     {
       name = n;
       age = a;
     }
 };

In above example, the Person class has two private properties (name and age) and a public constructor that takes two arguments (n for name and a for age). The constructor initializes the name and age properties of the object using the values passed as arguments.

When an object of the Person class is created, the constructor is automatically called with the specified arguments. For example, you could create a Person object like this: 

Person john("John Doe", 30);

In above example, the john object is created with the name "John Doe" and age 30. The constructor is called automatically with these arguments to initialize the name and age properties of the object.

Constructors can also have default values for their parameters, allowing you to create objects with default values for some properties. Here's an example of a constructor with default parameter values: 

 class Person {
   private:
     string name;
     int age;
   public:
     Person(string n = "", int a = 0)
     {
       name = n;
       age = a;
     }
 };

In above example, the Person constructor takes two parameters with default values (n defaults to an empty string and a defaults to 0). This allows you to create Person objects with default values for the name and age properties: 

Person jane;

Type of Constructors

In C++, there are several types of constructors that can be defined in a class:

Default constructor: A default constructor is a constructor that takes no arguments. If a class does not have any constructors defined, the compiler will automatically generate a default constructor. This constructor initializes all properties of the object to their default values.

Parameterized constructor: A parameterized constructor is a constructor that takes one or more arguments. This constructor is used to initialize the properties of the object with the values passed as arguments.

Copy constructor: A copy constructor is a constructor that takes an object of the same class as an argument. This constructor is used to create a new object with the same properties as the argument object.

Here's an example of each type of constructor:

 class Person {
   private:
     string name;
     int age;
   public:
     // Default constructor
     Person() {
       name = "";
       age = 0;
     }
 
     // Parameterized constructor
     Person(string n, int a) {
       name = n;
       age = a;
     }
 
     // Copy constructor
     Person(const Person& other) {
       name = other.name;
       age = other.age;
     }
 };

In above example, the Person class has a default constructor that initializes the name and age properties to their default values, a parameterized constructor that takes a string and an integer as arguments to initialize the name and age properties, a copy constructor that takes a const reference to another Person object and copies its name and age properties, and a move constructor that takes an rvalue reference to another Person object and moves its name and age properties into a new object.

Destructors

In C++, a destructor is a special member function of a class that is called automatically when an object of that class is destroyed. The destructor is used to clean up any resources that were allocated by the object during its lifetime.

Here's an example of a class with a destructor:

 class Person {
   private:
     string* name;
   public:
     Person(string n) {
       name = new string(n);
     }
     ~Person() {
       delete name;
     }
 };

In above example, the Person class has a private property name, which is a pointer to a string. The public constructor of the class takes a string argument n, allocates memory for the name pointer using the new operator, and initializes it with the value of n.

The class also has a destructor, which is called automatically when an object of the class is destroyed. The destructor deallocates the memory that was allocated for the name pointer using the delete operator.

When an object of the Person class is created, memory is allocated for the name pointer using the new operator. When the object is destroyed, the destructor is called automatically and deallocates the memory using the delete operator.

It's important to note that if a class does not have a destructor, the compiler will generate a default destructor that does nothing. However, if a class allocates any resources during its lifetime, it's important to define a destructor to properly deallocate those resources.

Also, it's worth noting that a destructor is only called once for each object, when the object is destroyed. If an object is created on the stack, the destructor is called automatically when the object goes out of scope. If an object is created on the heap using the new operator, the destructor must be called manually using the delete operation.

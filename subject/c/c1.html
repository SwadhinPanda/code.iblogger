How to Code?

  • Start by including the necessary header files, including stdio.h which provides input/output functions in C.
  • Declare three integer variables num1, num2, and num3 to store the three numbers to be inputted.
  • Ask the user to enter three numbers and then use scanf() function to read the three numbers into the variables
  • We use if-else statements to check which of the three numbers is the greatest. The if condition checks if num1 is greater than both num2 and num3, if true it prints num1 as the greatest number. Similarly, the else-if condition checks if num2 is greater than both num1 and num3, and prints num2 as the greatest number. If both of the above conditions are false, then the program executes the else statement and prints num3 as the greatest number.
  • The program returns 0 to indicate successful completion of the program

              #include <stdio.h>

              int main() {
                  int num1, num2, num3;
                  printf("Enter three numbers: ");
                  scanf("%d %d %d", &num1, &num2, &num3);
                    
                  if (num1 > num2 && num1 > num3) {
                     printf("%d is the greatest number.", num1);
                    }
                  else if (num2 > num1 && num2 > num3) {
                      printf("%d is the greatest number.", num2);
                    }
                  else {
                        printf("%d is the greatest number.", num3);
                    }
                    
                  return 0;
                }

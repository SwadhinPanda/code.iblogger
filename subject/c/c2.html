How to Code?

  • Declare variables for the operator and the two operands. In this practical, we will use char op for the operator and float num1 and float num2 for the operands.
  • Ask the user to enter the operator (+, -, *, /) and two operands using the printf and scanf functions.
  • Use the switch statement to perform the appropriate arithmetic operation based on the operator entered by the user.
  • Inside the switch statement, use the case keyword followed by the operator to create a case for each arithmetic operation.
  • For each case, write the code to perform the appropriate arithmetic operation using the operands entered by the user.
  • Use the break keyword to exit each case and prevent the code from executing the other cases.
  • Add a default case to the switch statement to handle any invalid operators entered by the user. This case should display an error message to the user.
  • Run the program and test it with different operands and operators.
  • Execute the program and verify that it works as expected.

      #include <stdio.h>

      int main() {
      float num1, num2, result;
      char op;
                
      printf("Enter first number: ");
      scanf("%f", &num1);
                
      printf("Enter second number: ");
      scanf("%f", &num2);
                
      printf("Enter operator (+, -, *, /): ");
      scanf(" %c", &op);
                
      switch (op) {
        case '+':
         result = num1 + num2;
         printf("%.2f + %.2f = %.2f", num1, num2, result);
         break;
        
        case '-':
         result = num1 - num2;
         printf("%.2f - %.2f = %.2f", num1, num2, result);
         break;
                   
        case '*':
         result = num1 * num2;
         printf("%.2f * %.2f = %.2f", num1, num2, result);
         break;
                   
        case '/':
          if (num2 == 0) {
               printf("Error: Division by zero");
           } else {
               result = num1 / num2;
               printf("%.2f / %.2f = %.2f", num1, num2, result);
           }
          break;
        default:
           printf("Error: Invalid operator");
          break;
               }
          return 0;
        }

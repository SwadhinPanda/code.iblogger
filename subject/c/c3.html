  3. Write a Program to print the sum and product of digits of an integer

How to Code?

  • Start by including the necessary header files such as stdio.h
  • Declare variables needed for the program, such as the integer to be analyzed in num, variables to hold the sum and product of its digits as sum and pr, and variables to hold individual digits in digit
  • Ask the user to enter integer
  • Use While loop to extract each digit of the integer one by one, and update the sum and product accordingly
  • Finally, print the sum and product of the digits
    • Download .txt 
                
      #include <stdio.h>

    int main() {
    int num, digit, sum = 0, pr = 1;
    
    printf("Enter an integer: ");
    scanf("%d", &num);
    while (num > 0) {
         digit = num % 10;  
         sum += digit;      
         pr *= digit;  
         num /= 10;   }
    
    printf("Sum of digits: %d\n", sum);
    printf("Product of digits: %d\n", pr);
    
    return 0;
    }

             Enter an integer: 637

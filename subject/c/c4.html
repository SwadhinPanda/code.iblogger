How to Code?

  • Start by including the necessary header files stdio.h
  • Declare variables needed for the program, such as int num to store the number,rev=0 for calculating reverse and digit
  • Ask the user to enter integer
  • Use while loop to extract each digit of the integer and perform logics to find reverse of given number
  • Finally, print the reversed number

      #include <stdio.h>

      int main() {
      int num, rev = 0, digit;
    
      printf("Enter a number: ");
      scanf("%d", &num);
      while (num > 0) {
          digit = num % 10;    
          rev = reve * 10 + digit;  
          num /= 10;             
       }
    
      printf("Reversed number: %d", rev);
    
        return 0;
    }

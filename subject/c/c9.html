  3. Write a Program to compute the factors of a given number

How to Code?

  • Start by including the necessary header files
  • Declare any variables needed for the program, such as the integer to be analyzed, variables to hold the sum and product of its digits, and variables to hold individual digits
  • Ask the user to enter integer
  • Use a loop to extract each digit of the integer one by one, and update the sum and product accordingly
  • Finally, print the sum and product of the digits
    • Download .txt 
                
                  #include <stdio.h>

                  int main() {
                      int num;
                  
                      printf("Enter a number: ");
                      scanf("%d", &num);
                  
                      printf("Factors of %d are: ", num);
                  
                      for (int i = 1; i <= num; i++) {
                          if (num % i == 0) {
                              printf("%d ", i);
                          }
                      }
                  
                      return 0;
                    }

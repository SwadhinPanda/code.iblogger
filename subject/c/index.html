C is a general-purpose programming language that was first developed in the early 1970s by Dennis Ritchie at Bell Labs. It is widely used for system programming, developing operating systems, embedded systems, and other low-level programming tasks.

C has features such as data types, control structures, functions, and arrays. It is a compiled language, which means that the source code is translated into machine code by a compiler before it can be executed.

Some important concepts in C programming include variables, which are used to store values, and functions, which are reusable blocks of code that perform specific tasks. Control structures, such as loops and conditionals, are used to control the flow of execution in a program.

C has influenced many other programming languages, and is still widely used today, especially in operating system development and other low-level programming tasks.

Tokens in C

In C programming, a token is the smallest individual unit of a program. A 'C' program consists of various types of tokens, which can be classified into the following categories:

Keywords: Keywords are reserved words that have a specific meaning in the C language. Examples include if, else, for, while, return, and void.

Identifiers: Identifiers are names given to variables, functions, and other entities in a C program. An identifier can consist of letters, digits, and underscores. It must start with a letter or an underscore. Examples of identifiers include sum, i, j, and my_function.

Variables: Variables are used to store and manipulate data in C. They need to be declared with a data type. For example 

int age = 19; // integer type
float temperature; // floating type
char grade = 'A'; // character type

Constants: Constants are fixed values that cannot be changed during program execution. C supports various types of constants such as integer constants, floating-point constants, character constants, and string constants. Examples include 42, 3.14, 'a', and "Hello World".

Operators: Operators are symbols that perform mathematical or logical operations on operands. C supports various types of operators such as arithmetic operators, relational operators, logical operators, and bitwise operators. Examples include + - * / < > && || &

Special symbols: These are used to indicate the beginning or end of a statement, block of code, or function. Examples include , ; { } ( )

It is important to note that whitespace (spaces, tabs, and line breaks) is not considered a token in C programming.

Datatypes in C

Datatypes specify the type of data that a variable can hold. C supports a variety of data types, including:

int: These data types are used to store integer values, which can be positive, negative, or zero. C supports four integer data types: char (1 byte), short (2 bytes), int (2 or 4 bytes), and long (4 or 8 bytes).

float: These data types are used to store real numbers with decimal places. C supports two floating-point data types: float (4 bytes) and double (8 bytes).

char: The char data type is used to store a single character value, such as a letter or digit. It takes up 1 byte of memory.

In addition to these basic data types, C also allows for the creation of user-defined data types, such as structures and enums.

Operators in C

In C programming, operators are symbols or keywords that perform various operations on data. Operators in C can be classified into the following categories:

Arithmetic operators: These operators perform mathematical operations such as addition, subtraction, multiplication, division, and modulus (remainder). Examples of arithmetic operators in C include +, -, *, /, and %. 

int a = 10, b = 5, result;
result = a + b; // Addition
result = a - b; // Subtraction
result = a * b; // Multiplication
result = a / b; // Division

Relational operators: These operators compare two values and return a Boolean value of true or false. Examples of relational operators in C include <, >, <=, >=, ==, and !=. 

int x = 5, y = 10;
if (x == y) { /* Code */ }
if (x < y)  { /* Code */ }

Logical operators: These perform logical operations on Boolean values and return a Boolean value of true or false. Examples of logical operators in C include && (logical AND), || (logical OR), and ! (logical NOT). 

int p = 1, q = 0;
if (p && q) { /* Code */ } // AND
if (p || q) { /* Code */ } // OR

Bitwise operators: These operators perform operations on individual bits of binary numbers. Examples of bitwise operators in C include & (bitwise AND), | (bitwise OR), ^ (bitwise XOR), and ~ (bitwise NOT). 

int m = 5, n = 3;
int result;
result = m & n; // Bitwise AND
result = m | n; // Bitwise OR

Assignment operators: These operators are used to assign values to variables. Examples of assignment operators in C include =, +=, -=, *=, /=, and %=. 

int value = 10;
value += 5; // (value = value + 5;)

Increment and decrement operators: These are used to increment or decrement the value of a variable. Examples of these operators in C include ++ (increment), and -- (decrement). 

int count = 0;
count++; // Increment
count--; // Decrement

Conditional operator: The conditional operator is a shorthand way of writing an if-else statement. It takes three operands and returns one of two values based on a Boolean condition. The syntax for the conditional operator in C is condition ? value_if_true : value_if_false. 

int a = 5, b = 3;
int result = (a > b) ? a : b;
printf("The greater number is: %d\n", result);

Input-Output Operations

C provides functions for input and output operations. prinft is used for output, and scanf is used for input. For example 

int num;
printf("Enter a number: ");
scanf("%d", &num);
printf("You entered: %d\n", num);

Expression

An expression is a combination of variables, constants, and operators that evaluates to a value. For example, a+b is an expression.

Order of Expression:
(Precedence and Associativity)

Operators have different precedence levels, and their associativity determines how they are grouped when they have the same precedence. For example, multiplication (*) has higher precedence than addition (+).

Control Structures

In C programming, control structures are used to control the flow of a program. C provides several types of control structures, including:

Decision Making and Branching

Simple IF Statement executes a block of code if a condition is true. 

if (condition) {
// Code to execute if condition is true
}

Simple IF-ELSE Statement executes one block of code if a condition is true and another if it's false. 

if (condition) {
    // Code to execute if condition is true
} 
else {
    // Code to execute if condition is false
}

Nesting IF…ELSE Statement: Using IF…ELSE statements inside each other for more complex conditions. 

if (condition1) 
  {
    if (condition2) 
    {
        // Code to execute if both conditions are true
    } 
    else {
        // Code to execute if condition1 is true, but condition2 is false
    }
  } 
else {
    // Code to execute if condition1 is false
}

IF…ELSE Ladder Statement: A series of IF…ELSE IF statements for multiple conditions. 

if (condition1) {
    // Code to execute if condition1 is true
} 
else if (condition2) {
    // Code to execute if condition2 is true
}
else {
    // Code to execute if no conditions are true
}

Selection Control Structure:
(Switch Statement)

The switch statement allows you to select one of many code blocks to be executed based on the value of an expression 

int choice;
switch (choice) {
case 1:
    // Code for choice 1
    break;
case 2:
    // Code for choice 2
    break;
default:
    // Code if none of the cases match
}

Loops in C

In C programming, loops are used to execute a block of code repeatedly until a certain condition is met. C provides three types of loop statements: while loop, do-while loop, and for loop.

While Loop: The while loop executes a block of code as long as a specified condition is true 

int count = 0;
while (count < 5)
{
  printf("Count: %d\n", count);
  count++;
}

In this example, the loop continues to execute as long as count is less than 5. It increments count in each iteration

Do While Loop: The do...while loop is similar to the while loop, but it guarantees at least one execution of the block of code before checking the condition. 

int num;
do {
  printf("Enter a positive number: ");
  scanf("%d", &num);
} while (num <= 0);

In this example, the program asks the user for a positive number at least once before checking if the condition is met.

For Loop: The for loop is often used for iterating over a sequence of values. 

for (int i = 0; i < 5; i++) {
  printf("Value of i: %d\n", i);
}

The for loop includes three parts: initialization, condition, and increment. It provides a compact way to control looping.

Jumps in Loop:
The break statement is used to exit a loop prematurely, regardless of the loop's condition. It is often used when a specific condition is met inside the loop. 

for (int i = 0; i < 10; i++) {
  if (i == 5) {
    break; // Exit the loop when i reaches 5
  }
  printf("Value of i: %d\n", i);
}

The continue statement is used to skip the current iteration and move to the next iteration of the loop 

for (int i = 0; i < 5; i++) {
  if (i == 2) {
    continue; // Skip iteration when i is 2
  }
  printf("Value of i: %d\n", i);
}

Arrays in C

In C, an array is a collection of elements of the same data type, stored in contiguous memory locations. Each element in an array can be accessed by its index or position in the array, starting from 0.

Arrays are declared using the following syntax:

data_type array_name[array_size];

where "data_type" is the data type of the elements in the array, "array_name" is the name of the array, and "array_size" is the number of elements in the array.

For example, to declare an array of integers with 5 elements: 

int my_array[5];

To initialize the array with values, we can use the following syntax: 

data_type array_name[array_size] = {value1, value2, ..., valueN};

where value1, value2, and valueN are the initial values for the first, second, and Nth element of the array, respectively.

For example, to initialize an array of integers with the values 1, 2, 3, 4, and 5: 

int my_array[5] = {1, 2, 3, 4, 5};

We can access individual elements in an array using their index, like this: 

my_array[0] // first element in the array (value is 1)
my_array[1] // second element in the array (value is 2)
my_array[2] // third element in the array (value is 3)
// and so on...

The above are examples of one dimensional array.

Multidimentional Array:

int matrix[3][3] = 
{
  {1, 2, 3},
  {4, 5, 6},
  {7, 8, 9}
};

Character Arrays and Strings:

In C, strings are represented as arrays of characters (char arrays). They are null-terminated, meaning they end with a null character '\0' 

char greeting[20] = "Hello, World!";

Subscript and Pointer Representation of Array:

Arrays can be accessed using both subscript notation (array[index]) and pointer notation (*(array + index)). 

int numbers[5] = {1, 2, 3, 4, 5};
int x = numbers[2]; // Subscript notation (accessing the third element, which is 3)
int y = *(numbers + 2); // Pointer notation (same result as above)

Pointers in C

In C, a pointer is a variable that stores the memory address of another variable. Pointers are declared using the asterisk (*) symbol, and they are used to manipulate and access memory directly.

The syntax to declare a pointer in C is as follows:

data_type *pointer_name;

Here, "data_type" is the data type of the variable that the pointer will point to, and "pointer_name" is the name of the pointer.

For example, to declare a pointer to an integer variable:

int *my_pointer;

To assign a value to a pointer, we can use the address-of operator (&) to get the memory address of a variable: 

int my_variable = 42;
int *my_pointer = &my_variable;

This sets "my_pointer" to point to the memory address of "my_variable".

To access the value of the variable that a pointer is pointing to, we use the dereference operator (*): 

int my_variable = 42;
int *my_pointer = &my_variable;
printf("The value of my_variable is: %d\n", *my_pointer);

This will output: "The value of my_variable is: 42".

Pointers can be incremented or decremented to navigate through arrays or memory. 

int numbers[5] = {1, 2, 3, 4, 5};
int* ptr = numbers; // Pointer to the first element of the array
int thirdElement = *(ptr + 2); // Accesses the third element (3) using pointer arithmetic

Pointers can be used for a variety of purposes, such as dynamic memory allocation, passing parameters to functions by reference, and accessing memory-mapped hardware registers. However, they also require careful management to avoid bugs like null pointer dereference and memory leaks.

Storage Class

In the C programming language, storage class specifies the lifetime, scope, and visibility of variables and functions. There are four types of storage classes in C:

Auto: Variables declared with the "auto" keyword are automatic variables. These variables have a lifetime that is limited to the block in which they are declared. They are created when the block is entered and destroyed when the block is exited. 

void foo() {
  auto int x = 10;
}

Register: Variables declared with the "register" keyword are register variables. These variables are stored in the CPU registers instead of memory, which makes them faster to access. However, the use of register variables is limited, as there are only a limited number of CPU registers available. 

register int count = 0;

Static: Variables declared with the "static" keyword are static variables. These variables have a lifetime that is the entire duration of the program. They are only initialized once, at the beginning of the program, and retain their value between function calls. 

void increment() {
  static int counter = 0;
  counter++;
  printf("Counter: %d\n", counter);
}

Extern: Variables and functions declared with the "extern" keyword are external variables and functions. These variables and functions are defined in a separate source file and are made available to other source files by including their declarations with the "extern" keyword. 

// In one file (file1.c)
int globalVar = 5;
                  
// In another file (file2.c)
extern int globalVar; // Declaration

Scope Rules:

Local Scope: Variables declared within a function have local scope and are accessible only within that function

Global Scope: Variables declared outside of any function have global scope and can be accessed from any part of the program.

Function Scope: Parameters and local variables within a function have function scope and are only accessible within that function.

Functions

Functions in C are a fundamental part of the language that allow you to group a set of instructions together that can be called multiple times from different parts of a program. Functions in C have the following syntax: 

  return_type function_name(parameters) {
    // Function body
    // ...
    return value; // Optional return statement
}

Type of Functions

User-Defined Function:

int add(int a, int b) {
return a + b;
}

Library Function:

#include <stdio.h>
int main() {
  printf("Hello, World!\n");
  return 0;
}

return_type: This is the data type of the value that the function will return. If the function does not return a value, the return type is void.

function_name: This is the name of the function, which must be a valid C identifier. The name should be descriptive of what the function does.

parameters: These are the variables that the function expects to receive as input, separated by commas. If the function does not require any input, the parameter list can be left empty or void can be used.

Function body: This is the set of instructions that the function will execute when it is called. The body can contain any valid C statements, including other function calls.

return statement: This is an optional statement that returns a value from the function. If the return type is void, the return statement can be omitted.

Here's an example of a simple function that adds two integers and returns the result: 

   int add(int x, int y) {
     int sum = x + y;
     return sum;
 }

This function takes two integers as input, adds them together, and returns the result as an integer. It can be called from another part of the program like this: 

int a = 2;
int b = 3;
int result = add(a, b); // result will be 5

Function Arguments:

Structures in C

In C programming language, structures are used to define a user-defined data type. A structure is a collection of different data types under a single name. Each member of a structure can have a different data type.

The syntax for defining a structure is:

  struct structure_name {
    data_type member1;
    data_type member2;
    .
    .
    .
    data_type memberN;
 };

Here, "structure_name" is the name of the structure and member1, member2, ..., memberN are the members of the structure. Each member is declared with its data type.

Here's an example of a structure that represents a person's information: 

   struct person {
     char name[50];
     int age;
     float height;
 };

In above example, person is the name of the structure, and it has three members: name, age, and height. name is a character array of length 50, age is an integer, and height is a floating-point number.

Once a structure is defined, we can create variables of that structure type: 

struct person p1;

Here, p1 is a variable of type person. We can access its members using the dot (.) notation: 

strcpy(p1.name, "John");
p1.age = 30;
p1.height = 5.8;

Here, we are assigning values to the members of p1. We can also pass a structure as an argument to a function: 

   void print_person(struct person p) {
     printf("Name: %s\n", p.name);
     printf("Age: %d\n", p.age);
     printf("Height: %f\n", p.height);
 }

Here, we are defining a function that takes a person structure as an argument and prints its members. We can call this function like this: 

print_person(p1);

This will print the values of the members of p1

Nested Structure:

struct Address {
    char street[50];
    char city[50];
};
struct Person {
    char name[50];
    struct Address address;
};

Self-Referential Structure: A self-referential structure is a structure that contains a pointer to another instance of the same structure type. This is commonly used for linked lists and tree structures. 

struct Node {
  int data;
  struct Node *next;
};

Array of Structure:You can create arrays of structures to store multiple records of the same type. 

struct Student students[100];

Structures and FunctionsFunctions can accept structures as arguments and return structures.

  struct Point result;
  result.x = p1.x + p2.x;
  result.y = p1.y + p2.y;
  return result;
}

Union in C

In C programming language, a union is a user-defined data type that allows you to store different data types in the same memory location. In other words, a union is a variable that can hold different types of data at different times, but only one type at a time.

The syntax for defining a union is similar to that of a structure: 

  union union_name {
    data_type member1;
    data_type member2;
    .
    .
    .
    data_type memberN;
 };

Here, union_name is the name of the union, and member1, member2, ..., memberN are the members of the union, each with its data type. The size of the union is determined by the largest member.

Here's an example of a union that can store an integer, a float, and a character array: 

  union data {
    int i;
    float f;
    char str[20];
};

In above example, data is the name of the union, and it has three members: i, f, str. Each member has a different data type.

To use a union, we can create a variable of that union type: 

union data d;

Here, d is a variable of type data. We can assign a value to one member of the union and access it using another member: 

d.i = 10;
printf("d.i = %d\n", d.i); 
// Output: d.i = 10

d.f = 3.14;
printf("d.f = %f\n", d.f); 
// Output: d.f = 3.140000

strcpy(d.str, "hello");
printf("d.str = %s\n", d.str); 
// Output: d.str = hello

In above example, we are assigning a value to d.i and then printing it. We are then assigning a value to d.f and printing it. Finally, we are assigning a string to d.str and printing it.

It's important to note that only one member of a union can be accessed at a time. When a new value is assigned to one member, the value of the other members is lost. Also, it's important to be careful when accessing members of a union, as it can lead to unexpected behavior if the wrong member is accessed.

File Management

In C programming language, file management is done using the standard input/output library functions. These functions allow you to open, read, write, and close files.

The main steps involved in file management in C are:

Opening a file: To open a file, you can use the fopen() function, which takes two arguments: the name of the file and the mode in which to open it. The mode can be "r" for reading, "w" for writing (overwriting if the file already exists), "a" for appending, or "r+", "w+", or "a+" for reading and writing. 

FILE *fp;
fp = fopen("file.txt", "w");
if (fp == NULL) {
    printf("Error opening file\n");
    exit(1);
}

In above example, we are opening a file called 'file.txt' for writing. If the file does not exist, it will be created. If the file cannot be opened, the program will exit with an error message.

Reading and writing data: Once a file is opened, you can use the fscanf(), fprintf(), fgets(), and fputs() functions to read and write data to the file. 

fprintf(fp, "Hello, world!\n");
char buffer[100];
fgets(buffer, 100, fp);

In above example, we are using fprintf() to write the string "Hello, world!" to the file and then using fgets() to read a line from the file into a buffer.

Closing the file: When you are done with a file, you should close it using the fclose() function. 

fclose(fp);

In above example, we are closing the file that we opened earlier.

It's important to check for errors when opening and closing files, as well as when reading and writing data to files. Errors can occur if a file does not exist, if there is not enough disk space to write to a file, or if there is a problem with the disk itself.

Here's an example program that reads a file called input.txt and writes the contents to a file called output.txt: 

#include <stdio.h>
#include <stdlib.h>
int main() {
     FILE *in_fp, *out_fp;
     char buffer[100];
 
    in_fp = fopen("input.txt", "r");
    if (in_fp == NULL) {
        printf("Error opening input file\n");
        exit(1);
   }

    out_fp = fopen("output.txt", "w");
    if (out_fp == NULL) {
        printf("Error opening output file\n");
        exit(1);
    }

   while (fgets(buffer, 100, in_fp) != NULL) {
        fprintf(out_fp, "%s", buffer);
   }

    fclose(in_fp);
    fclose(out_fp);

     return 0;
   }
